# Experiments

This folder contains analysis code for self-report and gaze data.

## Structure

- `self_reports_analysis.ipynb`: main self-report analysis notebook
- `gaze_analysis_umap.ipynb`: gaze-analysis workflow notebook
- `gaze_tobiii_preprocessing.py`: gaze preprocessing and trial/AOI integration
- `gaze_heatmap_visualization.py`: gaze heatmap and trajectory plotting
- `gaze_plot_anova_posthoc.py`: omnibus plotting with ANOVA/post-hoc summaries
- `gaze_plot_pairwise_ttest.py`: pairwise t-test plotting utilities
- `gaze_ml_prediction.py`: ML prediction from features
- `data/`: stimuli dataset and the results
- `figures/`: figures
